# Tripleten web_project_around_auth

    A complete Full Stack project of an amazing and secure personal website.

## Project Overview

This project ventures into put together front-end techonology and back-end functionality to make better the page making it a bit more secure against hackers and separating the user posibilities: only the user can upload or delete cards, as well as edit the profile, and visitors can only watch the page and like the cards.

## Live Demo

🌐 **[Live Project](https://maggiegutierrez.github.io/web_project_around_auth/)**

## Features

- **Responsive Design**: Optimized for desktop (1280px), tablet (768px), and mobile (320px) viewports
- **Interactive Elements**: Smooth transitions
- **Cross-browser Compatibility**: Works across all modern web browsers
- **Semantic HTML**: Properly structured markup for accessibility
- **POO**: Following the properly JavaScript structure for a better understanding

## Design Specifications

The project follows a mobile-first approach with the following breakpoints:

- **Mobile**: 620px and up
- **Tablet**: 720px and up
- **Desktop**: 1024px and up

## Development Process

This project was developed following the BEM (Block Element Modifier) methodology for a better organization and includes:

Semantic HTML structure
Mobile-first responsive design
Optimized images and assets
Clean, maintainable code
Future Enhancements
Include user-generated content features
POO & NPM
Clean files structure

## Installation & Setup

Clone the repository:
bash
git clone https://github.com/maggiegutierrez/web_project_around_auth.git
Navigate to the project directory:
bash
cd web_project_around_auth
With NPM installed and working, write on terminal 'npm run dev'. Clicking the command button, click on the link it shows you. Enjoy.
